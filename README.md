## Overview

SafetyCast aggregates real-time safety data from 25+ authoritative sources into an interactive map, helping users make informed safety decisions.

## Features

- **Real-time hazard mapping** - Visualize weather alerts, natural disasters, crime, transit disruptions, and more
- **AI-powered safety guidance** - Get recommendations aligned with FEMA/Red Cross best practices
- **8-factor risk scoring algorithm** - Prioritizes threats by proximity, severity, casualties, and impact
- **Emergency facility finder** - Locate nearby hospitals, fire stations, and shelters
- **Multi-city comparison** - Monitor safety across multiple locations simultaneously

## Data Sources

- NOAA Weather Alerts
- FEMA Emergency Declarations
- USGS Earthquakes
- FDA Recalls
- Transit systems (NYC MTA, WMATA, MBTA, TfL, etc.)
- Local news aggregation (80+ cities worldwide)
- And 20+ more authoritative sources

## Tech Stack

- **Frontend:** React, TypeScript, Vite, Tailwind CSS, Leaflet
- **Backend:** Node.js, Express
- **Database:** PostgreSQL (Drizzle ORM)
- **AI:** OpenAI GPT-4o-mini
- **Deployment:** Replit
